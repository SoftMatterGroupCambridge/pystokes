MIMA¶

Mollified irreducible multipole approach (MIMA) to Stokesian hydrodynamics

Mima3D¶

class pystokes.mima.Mima3D¶
solve()¶

solve takes two numpy arrays v and fk. Here, fk is the Fourier transform of the force field on the grid and v is an empty arrray of same size. It return the updated value of v = f(1-k k/k^2)/k^2
