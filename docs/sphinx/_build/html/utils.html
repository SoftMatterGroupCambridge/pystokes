utils.gridXY()¶

Returns the grid in XY direction centered around zero …

Parameters

  • L (float) – Length of the grid

  • Ng (int) – Number of grid points

Examples

An example of creating grid

>>> import numpy as np, pystokes
>>> dim, L, Ng = 3, 10, 32
>>>  rr, vv = pystokes.utils.gridXY(dim, L, Ng)
